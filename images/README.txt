This directory needs to contain the following icon files:

1. icon16.png (16x16 pixels)
2. icon48.png (48x48 pixels)
3. icon128.png (128x128 pixels)

Create simple ticket or seat icons with BookMyShow's red color (#e01c24) on white background.

You also need to add a notification sound file named "notification.mp3" in the root directory of the extension.
